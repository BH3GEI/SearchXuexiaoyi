# 基于蓝叠和学小易的超星剪切板搜题工具

## 使用步骤：
### 1、在蓝叠模拟器开启adb，在其中运行学小易
### 2、运行scrcpy -s <设备名> 将手机投屏到PC
### 3、确保adb环境正常并adb connect 127.0.0.1到模拟器
### 4、安装必要的Python包之后运行本脚本
### 5、截图之后，框选内容就会自动在模拟器中的学小易中显示结果

## 环境说明：
####  蓝叠：pixel2XL，分辨率1920x1080竖屏，240dpi
####  需要预先配置好scrcpy和adb
####  需要预先申请百度ocr的api
